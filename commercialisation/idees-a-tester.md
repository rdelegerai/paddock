# Idées à tester

Ces idées ne sont pas toutes à lancer maintenant. Elles servent de réserve.

## Idées fortes à tester vite

### Cadeau offert par un club

Un conseil d'administration peut acheter une vidéo pour :

- remercier un ancien pilote ;
- honorer un bénévole ;
- marquer le départ d'un président ;
- animer une assemblée générale ;
- offrir un cadeau lors d'un repas de club.

Pourquoi c'est prometteur :

- le paiement peut venir du club ;
- la vidéo a un usage collectif ;
- le club peut devenir relais.

### Cadeau familial

Angle très direct :

> Offrir à son père, grand-père ou proche une vidéo de son parcours de pilote.

Moments :

- anniversaire ;
- fête des pères ;
- Noël ;
- départ à la retraite ;
- hommage.

### Packs collectifs après premières ventes

À ne pas lancer tout de suite, mais possible :

- 3 vidéos pour un club ;
- vidéo pour trois anciens pilotes lors d'une assemblée générale ;
- tarif club après validation du modèle.

## Canaux à tester

### Email direct

Canal numéro 1 pour démarrer.

### Facebook groupes

À tester avec prudence :

- demander aux administrateurs ;
- publier peu, mais bien ciblé ;
- privilégier les groupes VHC, VHRS, rallye historique, anciennes sportives, clubs régionaux.

### LinkedIn

Utile pour :

- présidents ;
- dirigeants d'associations ;
- garages ;
- organisateurs ;
- contacts professionnels de Renan.

### Petites annonces spécialisées

À tester seulement après les premiers retours :

- La Vie de l'Auto ;
- journaux ou newsletters de clubs ;
- annonces d'événements historiques.

### Terrain

À tester si opportunité :

- rassemblement local ;
- course historique ;
- réunion ASA ;
- club proche géographiquement.

## Objections probables

### "Je peux faire un diaporama moi-même"

Réponse :

> Oui, bien sûr. Ici l'intérêt est d'avoir un rendu plus préparé, avec un texte structuré et un montage soigné, sans devoir tout faire soi-même.

### "Je n'ai pas assez de photos"

Réponse :

> Il faut au minimum 10 photos. Avec 15 à 25, le résultat est généralement plus vivant.

### "Je ne sais pas quoi raconter"

Réponse :

> Le questionnaire guide les souvenirs : débuts, voitures, lieux, anecdotes, personnes à citer, message final.

### "Est-ce adapté à un club ?"

Réponse :

> Oui, la vidéo peut être utilisée dans le cadre du club. Pour une diffusion publique, le client doit simplement s'assurer d'avoir les autorisations nécessaires.

### "Peut-on avoir une facture ?"

Réponse :

> Oui, la facture est générée automatiquement après paiement.
