# Réseaux sociaux

Objectif : créer de la visibilité et de la crédibilité, sans attendre une grosse audience.

Références utiles :

- Google Business Profile : service gratuit pour rendre une activité visible sur Google Search et Google Maps.
- LinkedIn Company Page : page d'entreprise gratuite pour rendre l'activité identifiable.
- YouTube Shorts, Instagram Reels et Facebook Reels : formats courts adaptés à l'extrait vidéo.

## Priorité

1. Facebook.
2. LinkedIn.
3. Instagram.
4. YouTube Shorts.

## Facebook

Pourquoi :

- beaucoup de clubs et passionnés auto y sont présents ;
- les groupes de voitures anciennes, rallye historique, VHC, VHRS et clubs locaux sont actifs ;
- bon canal pour le bouche-à-oreille.

Approche :

- créer ou utiliser une page Souvenir de Paddock ;
- publier l'exemple vidéo ;
- demander poliment aux administrateurs de groupes si la publication est acceptée ;
- éviter le spam dans 20 groupes le même soir ;
- privilégier les groupes ciblés.

Posts à tester :

- "Une idée de cadeau pour un ancien pilote".
- "Votre club cherche une idée pour mettre un membre à l'honneur ?"
- "À partir de photos anciennes, je crée une vidéo souvenir d'environ une minute."

## LinkedIn

Pourquoi :

- utile pour joindre présidents, responsables associatifs, dirigeants de garages ou organisateurs ;
- plus adapté aux messages directs sobres ;
- donne un peu de crédibilité professionnelle.

Approche :

- créer une page ou utiliser ton profil personnel ;
- publier le lancement ;
- contacter directement les personnes identifiées ;
- ne pas vendre trop fort au premier message.

Message court :

> Je lance Souvenir de Paddock, un service de vidéos souvenir pour pilotes et anciens pilotes. Je pense que cela peut intéresser certains clubs ou conseils d'administration qui souhaitent offrir un hommage à un membre. Je peux vous envoyer le lien si vous pensez que cela peut être utile.

## Instagram

Pourquoi :

- bon pour montrer l'ambiance ;
- utile si les vidéos courtes fonctionnent ;
- moins direct pour vendre à des clubs, mais utile pour la marque.

Formats :

- extrait de la vidéo exemple ;
- avant/après : photos sources puis vidéo ;
- citation courte sur la mémoire des paddocks ;
- mini-post "idée cadeau".

Fréquence :

- 2 à 3 publications par semaine au départ ;
- republier aussi en story.

## YouTube Shorts

Pourquoi :

- hébergement naturel pour extraits vidéo courts ;
- peut créer une visibilité lente ;
- utile pour envoyer un lien propre à certains contacts.

Formats :

- extrait 20 à 40 secondes ;
- version verticale si possible ;
- titre clair : "Vidéo souvenir pour ancien pilote de Formule Ford historique".

## Google Business Profile

À envisager, mais pas urgent si l'activité est surtout en ligne.

Intérêt :

- rendre l'activité visible sur Google ;
- donner une fiche professionnelle ;
- afficher le site et l'email.

Attention :

- vérifier les règles Google pour les activités sans accueil physique du public.

## Règles de publication

À faire :

- montrer l'exemple vidéo ;
- parler de souvenir, famille, club, hommage ;
- demander des relais ;
- rester sobre ;
- poster régulièrement.

À éviter :

- trop parler de prix dans tous les posts ;
- publier exactement le même message partout ;
- forcer dans les groupes ;
- utiliser trop de hashtags génériques ;
- promettre une qualité cinéma.
