# Stratégie commerciale

## Positionnement

Souvenir de Paddock est un service artisanal de vidéo souvenir pour pilotes de compétition historique.

Le ton doit rester :

- sobre ;
- humain ;
- personnel ;
- orienté mémoire, famille, club et hommage ;
- peu technique.

À éviter :

- trop parler d'IA ;
- vendre une "application" ;
- promettre une vidéo spectaculaire ;
- multiplier les offres avant les premières ventes ;
- donner l'impression d'un service impersonnel ou automatisé.

## Cible de départ

La cible prioritaire n'est pas le grand public.

Priorité :

1. Clubs automobiles historiques.
2. Présidents, secrétaires, trésoriers et membres de conseil d'administration de clubs.
3. Pilotes ou anciens pilotes membres de ces clubs.
4. Familles de pilotes.
5. Organisateurs de rassemblements, épreuves historiques ou événements de clubs.

Pourquoi commencer par les clubs :

- ils regroupent directement les bonnes personnes ;
- ils connaissent les anciens pilotes ;
- ils peuvent relayer le service ;
- ils ont parfois un budget associatif ;
- ils cherchent des idées de cadeaux ou d'hommages ;
- ils donnent de la crédibilité.

## Angles commerciaux

### Angle 1 : souvenir familial

Pour les enfants, conjoints, proches ou amis d'un pilote.

Phrase simple :

> Une courte vidéo pour raconter son histoire de pilote et garder une trace familiale.

Moments adaptés :

- anniversaire ;
- départ à la retraite sportive ;
- fête familiale ;
- Noël ;
- fête des pères ;
- hommage à un ancien pilote.

### Angle 2 : cadeau de club

Pour un président, un bureau ou un conseil d'administration.

Phrase simple :

> Une idée de cadeau de club pour remercier un pilote, un bénévole ou une figure historique de l'association.

Moments adaptés :

- assemblée générale ;
- remise de prix ;
- repas de club ;
- anniversaire du club ;
- hommage à un membre ;
- départ d'un président, pilote ou bénévole important.

### Angle 3 : mise en valeur d'un ancien

Pour les clubs qui veulent transmettre leur histoire.

Phrase simple :

> Valoriser les anciens pilotes, les bénévoles et les figures du club avec une vidéo courte et partageable.

### Angle 4 : cadeau entre passionnés

Pour les groupes d'amis de paddock.

Phrase simple :

> Chacun apporte quelques photos, et la vidéo devient un souvenir collectif.

## Proposition de valeur

Ce que le client gagne :

- une trace propre de l'histoire du pilote ;
- un format court, facile à montrer ;
- un rendu plus soigné qu'un simple diaporama ;
- une facture pour les clubs ou associations ;
- un service simple : questionnaire, photos, paiement, livraison.

Ce qui est différenciant :

- niche très précise ;
- ton patrimonial et humain ;
- parcours pensé pour les non-techniciens ;
- facture automatique ;
- prix de lancement accessible.

## Priorité commerciale

Ne pas chercher à tout faire en même temps.

Ordre recommandé :

1. Prospection directe clubs et responsables.
2. Réseaux sociaux avec contenus courts.
3. Relances.
4. Témoignages et exemples.
5. SEO simple seulement ensuite.

SEO veut dire : rendre le site lisible par Google pour qu'il puisse ressortir sur des recherches comme "cadeau ancien pilote rallye".
